# 📊 Bootstrap Hypothesis Testing App

A flexible and interactive application for performing bootstrap hypothesis testing. This tool is designed for both learning and practical statistical analysis.

---


### 🔹 Hypothesis Testing Setting

- Supports both:
  - Univariate analysis
  - Bivariate analysis
    - Paired samples
    - Independent samples

- Available test statistics:
  - Mean
  - Median
  - Variance

---

### 🔹 Data Input Options

- ✍️ Manual data entry  
- 📁 CSV file upload  

---

### 🔹 Data Overview

- 📈 Visualizations:
  - Histograms
  - Boxplots  

- 📊 Summary statistics tables  

---

### 🔹 Hypothesis Configuration:

1. Null hypothesis value  
2. Alternative hypothesis:
   - Two-sided  
   - Greater  
   - Less  
3. Significance level (α)  
4. Number of bootstrap samples  
5. Test statistic (mean, median, variance)  
6. Random seed (for reproducibility)  

---

### 🔹 Output & Results

After running the bootstrap test, the app provides:

- ✅ P-value  
- 📏 Confidence intervals  
- 🧾 Summary of all selected parameters  

- 📊 Bootstrap distribution visualization, including:
  - Distribution of bootstrapped statistics  
  - Indication of:
    - Confidence intervals  
    - Null hypothesis value  

---

### 🔹 Downloads

Users can download:

- 📉 Bootstrap distribution plots  
- 📦 Bootstrap replications 
- 📊 Summary tables and EDA outputs
